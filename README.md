# AES Шифровальщик

Простое графическое приложение для шифрования и расшифрования текста с использованием алгоритма AES (режим EAX). Программа позволяет генерировать ключи шифрования, сохранять и загружать зашифрованные данные в формате JSON.

## Возможности

- **Генерация ключей**: Автоматическая генерация криптографически безопасных ключей AES-256.
- **Шифрование текста**: Шифрование введённого текста с использованием режима EAX.
- **Расшифрование**: Загрузка и расшифрование данных из JSON-файлов.
- **Сохранение результатов**: Сохранение ключа, зашифрованного текста, тега и вектора инициализации в JSON-файл.
- **Интуитивный интерфейс**: Две вкладки для шифрования и расшифрования с понятными элементами управления.

## Установка (Windows .exe)

### Вариант 1: Готовый исполняемый файл

1. **Скачайте файл `AES_Encryptor.exe`** из раздела [Releases](../../releases) или из папки с дистрибутивом.

2. **Поместите файл в удобную папку** (например, `C:\Program Files\AES Encryptor\`).

3. **Создайте ярлык на рабочем столе** (опционально):
   - Щёлкните правой кнопкой мыши по `AES_Encryptor.exe`
   - Выберите "Создать ярлык"
   - Перетащите ярлык на рабочий стол

4. **Запустите приложение**, дважды щёлкнув по `AES_Encryptor.exe` или его ярлыку.
